Değişkenler

Değişken Nedir?

Programcılıkta kullandığımız dilin izin verdiği veri tipleri çerçevesinde istediğimiz verileri, bilgileri saklamamıza sağlayan yapılara değişken denir.


Nasıl Kullanılır?

int sayi
=
5;

Değişkenler Yukarıdaki Gibi Kullanılır

Önce değişkenin türü belirlenir. Örnek olarak
  • int
  • float
  • String
Eğer bu değişken türünü belrtimez isek program çalışmaz. Çünkü değişken türleri değişkenin kimliğidir.
Ardından değişkenin adı belirlenir. Örnek olarak
  • sayi
  • isim
  • deger
Ama değişkenler değişken türleri adında değişken adı alamazlar. Örneğin
  • int
  • float
  • char
Bir de değişken adlarının başında sayılara yer verilmez. Örneğin
  • 1sayi
  • 4deger
  • 2isim
Son olarak da değişkenin değeri olmalıdır. Değişkenin değeri olmazsa o değişken ile bir iş yapamayız.
