Veri Tipleri

Java dilinde bir değişkenin belirli bir veri tipi olmalıdır.



Veri Tipi Nedir?

Veri Tipi programlama dilinde veri tipleri oluşturduğumuz değişkenlerin hangi bellekte ne kadar alan kaplayacağını ve türde olacağını belirlemek için kullanılmaktadır.

Java Dilindeki Temel Veri Tipleri Şunlardır

Veri Tipi Verinin Boyutu Değer Aralığı
byte 1 byte -128 ile 127 aralığındaki değerleri kullanır.
short 2 byte -32768 ile 32767 aralığındaki değerleri kullanır.
int 4 byte -2147483648 ile 2147483647 aralığındaki değerleri kullanır.
long 8 byte -9223372036854775808 ile 9223372036854775808 aralığındaki değerleri kullanır.
float 4 byte Ondalıklı sayıları virgülden sonra 7 basamağına kadar tutar.
double 8 byte Ondalıklı sayıları virgülden sonra 15 basamağına kadar tutar.
boolean 1 byte true ya da false değerlerini kullanır.
char 2 byte Karakter/Harf/Sayı ve ASCII değerlerini kullanır.

int
intNum
=
5
;
//int tipinde veri tipi

floatNum
=
3.14f
