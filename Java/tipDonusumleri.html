Tip Dönüşümleri

Veri tipleri konusunda kullanım amaçlarına göre farklı veri tipleri olduğunu görmüştük. Bu veri tipleri arasında bazen dönüştürme ihtiyacı duymaktayız. Diyelim ki kullanıcıdan bir veri aldık. Bu veri ile matematiksel işlemler yapmak istiyorsak, String tipinde aldığımız örnek bir veriyi int tipine ya da double, float gibi veri tiplerine dönüştürmemiz gerekebilir. Çünkü kullanıcının girdiği sayı, String tipinde alınır. Şimdi bu dönüşümleri görelim.

String tipinden int tipine dönüştürme

Bu dönüştürme işlemi için iki yol vardır. Bunlar parseInt ve valueOf metotlarıdır.

String
ornek
=
"5"
;

int
yeni1
=
Integer
.
parseInt(ornek)
;

int

Integer
.
valueOf(ornek)
;

valueOf diğerine göre yavaştır ve Integer tipinde bir nesne döndürür.

parseInt hızlı ve int tipinde bir veri döndürür.

Bu dönüştürme işlemini virgülden sonraya kalan sayıları yok etmek için kullanacağız. Çünkü int tipi ondalık sayı almıyor fakat double, virgüllü sayı almaktadır.
