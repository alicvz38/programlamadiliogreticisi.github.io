Break

Devam etmekte olan döngüde break komutu geldiğinde o döngüden çıkar.

# Burada i değerini break komutunu görene kadar 0'dan başlayarak alt alta yazdırır.

for
i
in
range(10)
if
i
==
5:
break

print(i)

Continue

Devam etmekte olan döngüde continue komutu geldiğinde o döngüyü geçip diğer döngüye kaldığı yerden devam eder.

# Burada i değerini 0'dan başlayarak cumtinue komutunu gördüğünde o döngüyü
# atlayarak alt alta 10'a kadar yazdırır.

for
i
in
range(10)
if
i
==
5:
continue

print(i)
