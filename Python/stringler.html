Stringler

Python dilinde Stringler tek tırnak ya da çift tırnak içinde kullanılabilir.

print(
"Merhaba"
)
print(
'Merhaba'
)

Bir String'i Bir Değişkene Atama

a
=
"Merhaba"

print(a)

Çok Satırlı Metin Kullanımı

Üç tırnak arasına alt alta yazarak çok satırlı metinler kullanılır.

a
=

"""Merhaba Dunya
Bu Bir
Çok Satırlı Metin Denemesidir"""

print(a)

Bir String Üzerinden Döngü Kullanmak

Stringler dizi olduğundan karakterler arasında geçiş yapabilmek için for döngüsü kullanırız.

// Burada harfleri alt alta yazdırır

for x in Merhaba
:

print(x)

String Uzunluğu

String'lerin uzunluğunu bulmak için len() kullanabiliriz.

a
=
"Merhaba"

print(len(a))

Harfleri Büyük veya Küçük Yapma

Harfleri büyük yapmak için upper() ve küçük yapmak için de lower() kullanırız.

a
=
Merhaba

// MERHABA diye yazdırır.

print(a
.
upper())
// merhaba diye yazdırır.

print(a
.
lower())

Boşlukları Kaldırma

Boşlukları kaldırmak için strip() kullanırız.

a
=
" Merhaba "

print(a
.
strip())
// " Merhaba " dan "Merhaba" ya dönüşür

String'de Değişiklik Yapma

String'de değişiklik yapmak için replace() kullanırız.

a
=
"Merhaba"

print(a
.
replace(
"M"
, "H"))
// Ekran çıktısı olarak Herhaba verir.

İki String'i Toplama

İki String'i toplamak için + kullanırız.

a
=
"Merhaba"

b
=
"Dunya"

c
=
a
+
b
print(c)
// Ekran çıktısı olarak MerhabaDunya verir.

Aralarına boşluk koymak için " " kullanırız.

a
=
"Merhaba"

b
=
"Dunya"

c
=
a
+
" "
+
b
print(c)
// Ekran çıktısı olarak Merhaba Dunya verir.
