if - else if - else

if -else if -else koşul gerektiren durumlarda kullanılır. Koşulların durumları şu şekildedir.

Durum Açıklama
a > b Büyüktür
a >= b Büyüktür veya eşittir
a < b Küçüktür
a <= b Küçüktür veya eşittir
a == b Eşittir
a != b eşit değildir
