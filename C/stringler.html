Stringler

Stringler, metin/karakter depolamak için kullanılır.

Diğer birçok programlama dilinden farklı olarak, C, string değişkenlerini kolayca oluşturmak için bir string türüne sahip değildir. Ancak, C'de bir string oluşturmak için char türünü kullanırız ve bu karakterleri de bir dizi'de tutarız. Örneğin

char
metin
=
"Merhaba Dunya"
;

printf(
"%s"
, metin)
;

Metni kullanırken de çift tırnak kullanmalıyız.

Çıktı için de printf() içinde "%s" kullanmalıyız.

Erişim Stringleri

Stringler aslında C'deki dizilerle, bir Stringe köşeli parantez içindeki dizi'nin numarasına bakarak erişebiliriz. Örneğin

char
metin
=
"Merhaba Dunya"
;

prnitf(
"%c"
, metin
[
5
]
)
;

Ekran Çıktısı

b

Tek bir karakter kullanmak için de "%c" kullanırız.

String Değiştirme

Bir String'deki belirli bir karakterin değerini değiştirmek için dizinin numarası brlirlenir ve karakteri değiştirirken de tek tırnak kullanılır.

char
metin
=
"Merhaba Dunya"
;

metin[0]
=
'A'
;

// Burada Aerhaba Dunya çıktısı alırız.

String Uzunluğunu Alma

String uzunluğunu almak için sizeof() kullanırız. Örneğin

char
metin
=
"Merhaba Dunya"
;

printf(
"Metnin boyutu : %lu"
. sizeof(metin))
;

// 13 ve yukarısı olduğu için "%lu" kullandık. Çıktı olarak da 13 verir.
