Math Kütüphanesi

Bu dersimizde temel olarak kullanılan math kütüphanesi dersi anlatılacaktır. C dili de bu yapıya uyar. Tek fark C dilinde kütüphaneyi eklerken #include <math.h> kullanılırken. C++ dilinde ise sonunda h harfi olmadan #include <math> bu şekilde kullanılır. Örneklerimizle başlayalım.

sinDeger
=
sin(60)
;

// Burada cos(30) değerinin radyan karşılığı gelir.
