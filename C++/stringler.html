Stringler

Stringler metin depolamak için kullanılır.

Bir string değişken, çift tırnak içine alınmış bir karakter koleksiyonu içerir.

string
veri
=
"Merhaba"
;

Stringleri kullanabilmemiz için <string> kütüphanesine ihtiyacımız vardır.

// string kullanmamız için gereken kütüphane

#include <string>


string
veri
=
"Merhaba"
;

İki String'i Birletştirme

İki string'i birleştirmek için normal toplamadali gibi aralarına + koyarak yapabiliriz.

string
fName
=
"Merhaba"
;

string
lName
=
"Dunya"
;

string
full
=
fName
+
lName
;

cout
<<
full
;

Böyle yazdığımız zaman ekran çıktısında MerhabaDunya olarak çıktı verir. Burada boşluk koymak için de " " veya ' ' kullanırız.

string
fName
=
"Merhaba"
;

string
lName
=
"Dunya"
;

string
full
=
fName
+
" "
+
lName
;

// Ya da şu şekilde

string
full
=
fName
+
' '
+
lName
;

cout
<<
full
;
